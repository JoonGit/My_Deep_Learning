22.11.04 데이터 분석 내역
1번 상점 1월 일별 판매내역을 확인
*특이사항
- 1월 1일은 판매 내역이 없다 = 휴무
- 월별 판매 내역은 퍼센트 기준으로 비슷하다
- 1월 18일 경우 행사가 있어 가금류 판매량이 올라갔다

특이사항을 기점으로 데이터 분석을 한 결과
판매량은 항상 일정하다
하지만 변수(공휴일, 행사날)의 의해 변동된다

모델 학습을 위한 데이터 구축은
행사 없는날 기본으로 하며
행사 있는날에 값을 %로 값을 더해주는 데이터 구축
ex) 1월 2일(평일) = 10000
1월 3일(행사) = 10000 + a
1월 4일(행사, 오일)  = 10000 + a + b
이러한 데이터를 통해 판매량을 예측한다
